# UI Automation Framework (Java + Selenium + TestNG)

## Overview
This repository contains a **Java-based UI automation framework** built using **Selenium WebDriver, TestNG, and Maven**.  
The framework follows **real-world QA automation best practices** and is designed to be **scalable, maintainable, and CI-ready**.

The application under test is **SauceDemo**, a commonly used demo e-commerce site for automation practice.

This project is intended as a **portfolio-quality automation framework**, not a simple demo.

---

## Tech Stack
- **Language:** Java 17  
- **Automation Tool:** Selenium WebDriver  
- **Test Framework:** TestNG  
- **Build Tool:** Maven  
- **Driver Management:** WebDriverManager  
- **Logging:** SLF4J  
- **IDE:** IntelliJ IDEA
